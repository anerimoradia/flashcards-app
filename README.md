# JavaScript Flashcards

A clean, interactive flashcards app built with React.<br>
Flip cards, track progress, and study JavaScript concepts in a simple, elegant UI.

## Overview
This project is part of the **Frontend Projects** track on roadmap.sh. <br>
It focuses on building a simple yet interactive flashcard interface that helps users study JavaScript concepts.

The app includes:

- A 3D flip animation  
- Next/Previous navigation  
- A dynamic progress bar
